## ghost-mini-program
基于ghost api开发的微信小程序
> ghost版本2.9, ghost api版本v0.1

## ⚙️ 安装
```
$ git clone https://github.com/gitHber/ghost-mini-program.git
$ npm install
```
## ⚙️配置
1. 配置你的appid
> 修改project.config.json为你的appid
2. 配置你的ghost
> 修改ghost-config.js中的配置，可以打开你的网址，控制台下输入ghost.url.api()就能获取到你的配置了
